# Retail Sales Performance Dashboard

## 📊 Project Overview
This project analyzes retail sales data (5000 records) using Power BI.

The dashboard provides insights into:
- Total Sales
- Total Profit
- Profit Margin
- Monthly Sales Trend
- Sales by City
- Sales by Category
- Top 5 Products
- Payment Method Analysis

## 🛠 Tools Used
- Power BI
- Excel
- DAX
- Data Visualization

## 🎯 Key Insights
- Total Sales: 135M+
- Profit Margin: 43%
- Highest Sales City: Chennai
- Most Used Payment Mode: Credit Card
